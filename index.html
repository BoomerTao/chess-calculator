How this works

Norm ladder uses the official US Chess norm coefficient, not Elo. For title level Y and an opponent's post-event rating Rᵢ, let Δ = Y − Rᵢ; then Cᵢ = 0 if Δ ≤ −400, 0.5 + Δ/800 if −400 < Δ ≤ 0, 0.5 + Δ/400 if 0 < Δ ≤ 200, and 1 if Δ > 200. Sum to get CT = Σ Cᵢ. A norm is earned when the score strictly exceeds CT + 1.0 (or + 2.0 if a 3-game event had a 4th-round bye/forfeit). The comparison is full-precision; values are only rounded for display.

Points-needed diagnostic — when the top achievable norm is missed, a counterfactual estimate of how much stronger the (fixed) field would have needed to be: the uniform gap per opponent and the cheapest single-opponent bump, found by binary search since CT decreases monotonically with opponent ratings.

Rating projection uses the official US Chess rating formula. N* = 50 / √(0.662 + 0.00000739 × (2569 − R)²) for R ≤ 2355 (else 50); Ne = min(N, N*); K = 800 / (Ne + m); We = Σ Elo-expected(R, Rᵢ); bonus = max(0, K(S − We) − 10√max(m,4)) when m ≥ 3; new rating = R + K(S − We) + bonus. Class floors at 1200–2100; peak-based floor = largest class floor at or below (peak − 200).

FIDE uses its own expected-score (a logistic approximation of the official lookup table, §8.1.2) with the 400-point rating-difference cap for players below 2650 (§8.3.1). K = 40 / 20 / 10 per §8.3.3, with the K×n ≤ 700 cap; change rounded away from zero (§8.3.4); below 1400 shown unrated (§7.2.1). FIDE Handbook B.02, effective 1 March 2024.

Performance rating is the single rating at which expected score equals the actual score, solved numerically (uncapped logistic). Undefined for a clean sweep or total loss.
