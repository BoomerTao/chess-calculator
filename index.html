How this works

Shared helper: expectedScore(playerRating, opponentRating) = 1 / (1 + 10^((opp − player) / 400)). This is the standard logistic formula used by both panels.

Norm panel — for each title level Y, expectedTotal(Y) sums expectedScore(Y, Ri) over all opponents (the hypothetical Y-rated player's expected score vs. this field). A norm at level Y requires your actual score S to be strictly greater than expectedTotal(Y) + 1.00. Since chess scores are multiples of 0.5, the minimum qualifying score is minScore = (⌊(expectedTotal + 1.0) / 0.5⌋ + 1) × 0.5. Verify: expected = 2.5 → threshold = 3.5 → minScore = 4.0.

Rating panel (Glickman & Doan, April 6, 2026 spec, §3–4.2):

• N* = 50 / √(0.662 + 0.00000739 × (2569 − R)²) if R ≤ 2355, else 50

• Ne = min(N, N*) — effective game count

• K = 800 / (Ne + m)

• We = Σ expectedScore(R, Ri) over all opponents

• Base change = K × (S − We)

• Bonus (when m ≥ 3 and all opponents distinct): m* = max(m, 4); bonus = max(0, K(S−We) − B√m*) where B = 10 (effective Jan 1, 2025)

• R_new = R + K(S − We) + bonus, rounded to nearest integer

Rating floors (§5): Class floors at 1200, 1300, …, 2100. Peak-based floor = largest class floor ≤ (peak − 200). The absolute floor is 100.
