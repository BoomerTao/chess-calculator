How this works

Shared helper: expectedScore(playerRating, oppRating) = 1 / (1 + 10^((opp − player) / 400)) — the standard logistic formula used by both panels.

Norm panel — for each title level Y, expectedTotal(Y) sums expectedScore(Y, Rᵢ) over all opponents. A norm requires your actual score S to be strictly greater than expectedTotal + 1.00. Since chess scores are multiples of 0.5, the minimum qualifying score is minScore = (⌊(expectedTotal + 1.0) / 0.5⌋ + 1) × 0.5. Verify: expected = 2.5 → threshold = 3.5 → minScore = 4.0.

Rating panel — based on the official US Chess rating system specification:

• N* = 50 / √(0.662 + 0.00000739 × (2569 − R)²) if R ≤ 2355, else 50

• Ne = min(N, N*) — effective game count (caps at 50; lower for lower-rated players)

• K = 800 / (Ne + m)

• We = Σ expectedScore(R, Rᵢ) over all opponents

• Bonus (m ≥ 3, distinct opponents): m* = max(m, 4); bonus = max(0, K(S−We) − 10√m*)

• R_new = R + K(S − We) + bonus, rounded to nearest integer

Floors: class floors at 1200, 1300, …, 2100. Peak-based floor = largest class floor at or below (peak − 200).
